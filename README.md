# Grade Management System

## Project Overview
This is a simple Java-based Grade Management System that allows storing student marks, calculating averages, assigning grades, and generating performance reports. It uses arrays and menu-driven operations for easy interaction.

## Features
- Add student marks for multiple subjects
- View all student marks
- Calculate averages and grades
- Generate performance reports
- Menu-driven interface
- Input validation for marks (0-100)

## Technical Details
- Language: Java
- Data Storage: Arrays (student names and 2D array for marks)
- Subjects: Mathematics, Science, English, History, Computer
- Grade Categories: A+, A, B, C, D, F
